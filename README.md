

## Virtual Environment
Install virtualenv

```python
python3 -m pip install virtualenv
```

check version
```python
virtualenv --version
```

 create virtual environtment

 ```python
 virtualenv ml_package
 ```

 activate virtual environment

 for linux/mac
 ```python
 source ml_package/bin/activate
 ```
 for windows 
 ```python
 ml_package\Scripts\activate
 ```
